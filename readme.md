## ✨ 功能特性

### 🎨 极致的视觉体验
- **macOS 风格设计**: 像素级复刻原生体验，简洁优雅。
- **自动深色模式**: 随系统自动切换亮色/暗色主题。
- **流畅动画**: 精细打磨的微交互和列表动画。

### ⚡️ 高效下载管理
- **实时监控**: 动态显示下载进度、实时速度和剩余时间。
- **智能操作**: 支持暂停、取消、重试下载，一键打开文件或所在文件夹。
- **图标徽章**: 浏览器工具栏图标实时显示正在进行的下载任务数。

### 🆕 独家特色功能
- **新建下载**: 支持手动输入 URL 创建新的下载任务。
- **延迟删除与撤销**: 误删记录？别担心！移除记录后有 **5秒反悔期**，一键撤销恢复。
- **智能清理**: 明确区分 **"仅移除记录"** 与 **"删除本地文件"**，防止误删重要数据。

### 🔍 强大的筛选与搜索
- **实时搜索**: 毫秒级响应，支持搜索文件名或下载来源域名。
- **快捷键支持**: 按 `/` 键快速聚焦搜索框，`Esc` 退出。
- **类型过滤**: 自动分类文档、图片、压缩包、安装包等，查找文件快人一步。

## 🚀 快速开始

### 安装说明
1. 克隆/下载本项目到本地。
2. 在 Chrome 地址栏输入 `chrome://extensions/`。
3. 开启右上角的 **"开发者模式"**。
4. 点击 **"加载已解压的扩展程序"**。
5. 选择项目根目录即可。

### 🛠 使用技巧
- **搜索**: 按下 `/` 键即可快速开始搜索。
- **批量操作**: 点击右上角菜单可一键清理所有已完成或失败的项目。
- **撤销删除**: 移除某条记录后，留意底部的提示条，点击“撤销”即可恢复。

## ⚙️ 技术栈

- **Manifest V3**: 最新一代 Chrome 扩展标准，更安全、更省电。
- **Vanilla JS & CSS**: 无繁重框架依赖，极速启动，原生 CSS 变量实现主题切换。
- **Native APIs**: 深度集成 `chrome.downloads`, `chrome.storage`, `chrome.action`。
